# Enable GitHub Pages - Quick Guide

Your website is ready! Just enable GitHub Pages to make it live.

## Steps to Enable GitHub Pages:

1. **Go to your repository on GitHub:**
   - Visit: https://github.com/rakarakaka/Portofolio_DanNaraka

2. **Open Settings:**
   - Click on the **"Settings"** tab (at the top of your repository)

3. **Go to Pages:**
   - In the left sidebar, click **"Pages"**

4. **Configure Source:**
   - Under **"Source"**, select:
     - **Branch:** `main`
     - **Folder:** `/ (root)`
   - Click **"Save"**

5. **Wait for Deployment:**
   - GitHub will build and deploy your site (takes 1-2 minutes)
   - You'll see a green checkmark when it's ready

6. **Access Your Live Website:**
   - Your site will be available at:
   - **https://rakarakaka.github.io/Portofolio_DanNaraka/**
